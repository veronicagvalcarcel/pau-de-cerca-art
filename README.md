# Pau de cerca art

A self-managed, self-taught art project exploring manual and artisanal crafts with a social and dissident perspective. This is more than a project—it's a form of expression and exploration.

## ✨ Features

- **Product Showcase**: Browse collections of camisetas (t-shirts), prints, and totebags
- **Product Detail View**: Explore each product with high-quality images and detailed information
- **Responsive Design**: Fully responsive experience across all devices
- **Smooth Animations**: Motion effects for transitions and interactions
- **Instagram Integration**: Direct links to social media and embedded feed
- **About Section**: Learn about the artist and the project

## 🛠️ Tech Stack

- **React 18** with TypeScript
- **Vite** - Next generation frontend build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Motion/Framer Motion** - Animation library
- **Lucide React** - Icon library

## 📋 Prerequisites

- Node.js (v16 or higher)
- npm or yarn

## 🚀 Getting Started

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server (http://localhost:3000)
npm run dev
```

### Build & Preview

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

### Maintenance

```bash
# Type checking and linting
npm run lint

# Clean build output
npm run clean
```

## 📦 Collections

- **Camisetas** - T-shirts in various colors and designs (S to 4XL)
- **Prints** - High-quality prints in A5 and A4 sizes
- **Totebags** - Cotton tote bags (38x42 cm)

## 🎨 Customization

Product data and images are sourced from `src/constants.ts`. To add new products:

1. Update product arrays in `src/constants.ts`
2. Place corresponding images in `public/escaparate/{category}/`
3. Follow the naming convention: `{category_prefix}{id}.webp`
